About HashPath
Design Philosophy
HashPath is a single HTML file you can drop on a USB stick. No build system, no frameworks, no servers, no accounts, no tracking. Open it in any browser, on any OS.

Your inputs and saved scenarios live in your browser's local storage and never leave your device. The only outbound calls are to CoinGecko for BTC price and mempool.space for network difficulty and the halving countdown. Back up by exporting JSON; migrate by copying the HTML file and your export.

When served over HTTPS, an optional service worker (sw.js) caches the calculator for offline use. Install as a PWA from Settings.

Simple. Private. Durable.
